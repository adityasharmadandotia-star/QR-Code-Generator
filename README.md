# 🔳 QR Code Generator

A simple and beginner-friendly **QR Code Generator** built with **Python**. This application allows users to generate QR codes for any text or URL and save them as PNG images.

---

## ✨ Features

* 🔗 Generate QR codes for any text or URL
* 🖼️ Save QR codes as PNG images
* ⚡ Fast and easy to use
* 🐍 Built with Python
* 👨‍💻 Beginner-friendly project

---

## 🛠️ Technologies Used

* Python 3
* qrcode
* Pillow (PIL)

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/adityasharmadandotia-star/QR-Code-Generator.git
```

Install the required library:

```bash
pip install qrcode[pil]
```

---

## ▶️ Usage

Run the program:

```bash
python qr_generator.py
```

Enter any text or URL, choose a file name, and your QR code will be saved as a PNG image.
